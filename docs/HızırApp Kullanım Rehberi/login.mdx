# HızırApp'e Giriş Yapmak



Uygulamaya daha önce kaydolup giriş yaptıysanız bilgilerinizi girdikten sonra `giriş yap` butonuna tıklayarak uygulamayı kullanmaya başlayabilirsiniz. Henüz kayıt olmadıysanız ekranın altında bulunan `kayıt ol` butonu aracılığıyla kayıt olma ekranına geçiş yaparak sizden istenen bilgileri veriniz.
